# Logo

Placez ici le logo de N'SAPKA (main stylisée).

## Format recommandé
- Format : PNG avec fond transparent
- Tailles : 
  - `logo_512.png` - 512x512px
  - `logo_256.png` - 256x256px
  - `logo_128.png` - 128x128px
  - `logo_64.png` - 64x64px

## Note
Actuellement, l'application utilise l'icône Material `Icons.back_hand`.
Remplacez-la par votre logo personnalisé.
