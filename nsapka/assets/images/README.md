# Images

Placez ici les images de produits artisanaux.

## Format recommandé
- Format : PNG, JPG
- Taille : 800x800px minimum
- Poids : < 500KB par image

## Exemples
- `onboarding1.png` - Image pour la première page d'onboarding
- `onboarding2.png` - Image pour la deuxième page d'onboarding
- `onboarding3.png` - Image pour la troisième page d'onboarding
- `product_*.png` - Images de produits
